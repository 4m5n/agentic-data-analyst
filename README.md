The Agentic Data Analyst backend is a Python-based system that enables users to upload raw CSV data and ask business questions in natural language, receiving answers grounded in real data analysis. Built with FastAPI, it combines a data cleaning pipeline, a set of modular analysis tools, and an LLM-driven agent that dynamically selects and executes the appropriate operations to generate insights. Instead of relying on static queries or dashboards, the system behaves like an automated analyst—cleaning messy data, performing calculations such as sales or profit trends, optionally generating visualizations, and returning concise, business-focused explanations. This approach reduces the manual effort typically required in data analysis workflows and provides a fast, scalable way to extract insights, making it a practical foundation for internal analytics tools, decision support systems, or AI-powered data applications.
