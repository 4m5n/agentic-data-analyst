The Agentic Business Analyst is a Python-based project that allows users to ask business questions about the Superstore dataset using natural language and receive answers grounded in real data analysis. Built with FastAPI, it combines a data cleaning pipeline, a set of modular analysis tools, and an LLM-driven agent that dynamically selects and executes the appropriate operations to generate insights. Instead of relying on predefined dashboards or static queries, the system behaves like an automated analyst—cleaning and structuring the dataset, performing calculations such as sales and profit trends, optionally generating visualizations, and returning concise, business-focused explanations. This approach simplifies the typical analytics workflow and demonstrates how agentic AI can be used to turn structured business data into actionable insights without manual querying or scripting.
